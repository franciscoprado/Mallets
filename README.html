Mallets is a Actionscript 3 library dedicated to contain a series of pre-made ??components and useful for games likepreloaders, UI components (like listboxes, scrollers, etc..), bitmap blitting, among others.

The idea for the creation of the library is to facilitate the reuse of common objects to create games. The library purpose is speed up the development, focusing repetitive tasks in a few lines of code. For example, there is already embedded graphics and the class to create a listbox. What you have to do alone is to create an object MlListBox and add their values??.

Features:

Currently, the library contains:
  • Assets preloader: gathering a lot of resources into a single preloader.
  • Bitmap Blitting with a easy way to manipulate.
  • Translation: create XML files with different languages ??and translate in a simple way.
  • Pathfinding embedded.
  • Sound: play and stop sounds through a single class.
  • AI: basic interface State and Finite State Machine class.
The repository can be found at GitHub: https://github.com/franciscoprado/Mallets
